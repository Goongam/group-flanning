## 프로젝트 소개
목표 달성을 위해 그룹을 만들며 계획을 추가하고 매일 수행했던 내용을 기록할 수 있는 웹 사이트 입니다.




## 개발기간
2022.7 ~ 2022.10



## 개발환경
Next.js 13

IDE: vscode

DB: mongoDB

server: vercel
